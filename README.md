# Mushroom Classifier

This project is a machine learning application that predicts whether a mushroom is edible or poisonous.

## Project Structure

- `Mushroom_classifier.ipynb`  
  Notebook used for data preprocessing, training and evaluation.

- `mushroom_model.pkl`  
  Trained machine learning model.

- `label_encoders.pkl`  
  Encoders used to transform categorical values.

- `mushroom.csv`  
  Dataset used for training.

- `app.py`  
  GUI application used to interact with the model.

## Technologies

- Python
- Scikit-learn
- Pandas
- Tkinter

## How to Run

### 1. Install dependencies

pip install pandas scikit-learn joblib

### 2. Run the application

python app.py

### 3. Use the GUI

The graphical interface will open and allow the user to select mushroom features and predict whether it is edible or poisonous.
